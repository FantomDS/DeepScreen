# DeepScreen
Учебный проект по Системной и программной инженерии

Откройте терминал и выполните:

```
Перейдите в папку проекта
cd deepscreen

Создание виртуального окружения
python -m venv venv

Активация
venv\Scripts\activate


Для Mac/Linux:

Создание виртуального окружения
python3 -m venv venv

Активация
source venv/bin/activate
